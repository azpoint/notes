# ⌨️ Keyboard Shortcuts

---

## 🧠 Autocompletion (TAB)

```text id="t8k4zm"
TAB            # Auto-complete command or filename (if unique)
TAB TAB        # Show all possible matches
```

---

## 🧹 Terminal Control

```bash id="p3v7qn"
CTRL + L       # Clear terminal screen
CTRL + D       # Exit shell / close terminal session
```

---

## ✂️ Line Editing

```bash id="y6k2lc"
CTRL + U       # Cut (delete) entire line before cursor
```

---

## 🧭 Cursor Navigation

```bash id="g9m3sr"
CTRL + A       # Move cursor to start of line
CTRL + E       # Move cursor to end of line
```

---

## 🛑 Process Control

```bash id="n5k4zd"
CTRL + C       # Terminate current running command
CTRL + Z       # Suspend (pause) running process
```

---

## 🖥️ Terminal Launch

```bash id="m7x2lc"
CTRL + ALT + T # Open a new terminal (common Linux desktop shortcut)
```
